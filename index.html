Linguistic-Feature-Matrix-Generator

This is a small web page used to create feature matrices for words. This is a useful tool in linguistics to define the behaviors of words either semantically or syntactically. To use this tool, go to this repo's GitHub Pages.

View the Project on GitHub SpencerWBarnes/Linguistic-Feature-Matrix-Generator

[ and ] to start or end a matrix bracket
, to add space between words in a row
Enter or ; to go down a line
not followed by a space to place a &not symbol
phi followed by a space to place a φ symbol
